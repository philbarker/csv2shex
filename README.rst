Csv2shex
========

Generate ShEx schemas from application profiles in CSV.

Documentation
-------------

See https://csv2shex.readthedocs.io/en/latest/ (not much there yet)

Installation
------------

.. code-block:: bash

   $ git clone https://github.com/tombaker/csv2shex.git
   $ cd csv2shex
   $ python -m venv .venv
   $ source .venv/bin/activate
   $ pip install flit
   $ flit install -s
